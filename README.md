Author

📖 Project Overview
This project is a comprehensive Sales Data Analysis Workbook developed to demonstrate practical Excel skills used in business reporting, data analysis, and dashboard development. The workbook contains a structured sales dataset along with various analytical exercises designed to transform raw transactional data into meaningful business insights.
The objective of this project is to strengthen analytical thinking, improve Excel proficiency, and simulate real-world business reporting scenarios.
🎯 Project Objectives
Analyze sales performance across different regions and product categories.
Identify top-performing products and customers.
Create meaningful dashboards and visual reports.
Apply advanced Excel formulas and functions.
Generate actionable business insights from raw sales data.
Practice data cleaning and transformation techniques.
